Geography Explorer - Site Plan

Site Name

Geography Explorer

This site will provide an interactive way to compare different locations based on population, landmarks, and fun facts.

Site Purpose

The website aims to help users explore and compare geographical locations worldwide using dynamic data and interactive features.

Scenarios

  • How does the population of two different cities compare?
  • What are the most interesting landmarks in a given country?

Color Schema

  • Fern Green (#4B7F52): Used for headings and accents.
  • Mindaro (#D2F898): Used for background highlights.

Typography

Font: Lato (Regular 400, Bold 700)

Used for both headings and body text to ensure readability and consistency.
